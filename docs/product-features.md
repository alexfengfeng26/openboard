# 产品功能说明

## 产品概述

**Kanban Board** 是一个现代化的看板管理系统,采用 Markdown 文件作为数据存储,支持 AI 智能辅助管理。

---

## 核心功能

### 1. 多看板管理
- **创建看板**: 支持创建多个独立的看板项目
- **编辑看板**: 修改看板标题等元信息
- **删除看板**: 删除不再需要的看板
- **看板切换**: 在看板之间快速切换

### 2. 列表(Lane)管理
- **创建列表**: 在看板中创建多个列表(如:待办、进行中、已完成)
- **编辑列表**: 修改列表标题
- **删除列表**: 删除列表及其中的所有卡片
- **列表排序**: 拖放调整列表顺序

### 3. 卡片(Card)管理
- **创建卡片**: 在指定列表中创建任务卡片
- **编辑卡片**: 修改卡片标题和详细描述
- **删除卡片**: 删除单张卡片
- **卡片排序**: 在同一列表内拖放排序
- **跨列表移动**: 将卡片拖动到其他列表

### 4. 标签系统
- **预设标签**: 每个看板可配置标签池(紧急、功能、Bug、优化、文档、设计)
- **标签颜色**: 支持自定义标签颜色(十六进制)
- **卡片标签**: 为卡片添加多个标签进行分类

### 5. 拖放交互
- **列表级排序**: 拖拽整个列表调整顺序
- **卡片级排序**: 在同列表内拖拽卡片调整优先级
- **跨列表移动**: 将卡片拖到不同列表改变状态
- **无障碍支持**: 支持键盘操作

### 6. AI 智能助手
- **自然语言操作**: 通过对话方式创建/编辑/删除卡片和列表
- **工具调用系统**: AI 可直接执行看板操作
- **智能推荐**: 根据上下文推荐操作
- **操作日志**: 记录所有 AI 执行的操作便于追溯

### 7. 实时更新
- **乐观更新**: UI 立即响应,无需等待服务器
- **失败回滚**: 操作失败时自动恢复原状
- **即时反馈**: Toast 通知显示操作结果

---

## 数据存储

- **存储格式**: Markdown 文件
- **文件位置**: `data/{board-id}.md`
- **并发安全**: 文件锁机制防止同时写入
- **迁移支持**: 自动从 JSON 格式迁移到 Markdown

---

## AI 工具能力

| 工具名称 | 功能描述 |
|---------|---------|
| create_card | 在指定列表中创建新卡片 |
| update_card | 更新卡片标题和描述 |
| move_card | 将卡片移动到其他列表 |
| delete_card | 删除指定卡片 |
| create_lane | 创建新列表 |
| update_lane | 更新列表标题 |
| delete_lane | 删除列表 |
| create_board | 创建新看板 |
| update_board | 更新看板信息 |
| delete_board | 删除看板 |

---

## 技术特点

- **服务端渲染**: Next.js App Router 提供 SSR 支持
- **类型安全**: TypeScript 提供类型检查
- **响应式设计**: Tailwind CSS 适配各种屏幕尺寸
- **现代化 UI**: Radix UI 提供无障碍组件
- **流畅动画**: 拖放操作带有平滑过渡效果

---

## 使用场景

- **个人任务管理**: 管理日常待办事项
- **团队协作**: 多人共享项目进度
- **敏捷开发**: Sprint 规划和任务跟踪
- **项目规划**: 可视化的项目阶段管理
